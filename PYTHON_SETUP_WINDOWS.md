Python Setup Windows
====================

Download Python 2.7.X Windows Installer from:

<https://www.python.org/download>

Add to path:

    C:\Python27\

Download get-pip.py from:

<https://bootstrap.pypa.io/get-pip.py>

Run:

```shell
python get-pip.py
```

Add to path:

    C:\Python27\Scripts\

Run:

```shell
pip install virtualenv
mkdir C:\virtualenvs
pip install ipython
```
